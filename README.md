The easiest way to preview your GitHub code as a live webpage is GitHub Pages — it's free and built right into GitHub.
Here's how to enable it:

Go to your repository on GitHub
Click Settings (top right of the repo)
Scroll down to Pages in the left sidebar
Under Source, select Deploy from a branch
Choose your branch (usually main or master) and click Save
GitHub will give you a URL like https://yourusername.github.io/your-repo-name
